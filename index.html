Tugas Pertama

  • Membuat akun dan repository di github.com
      • - Nama repository boleh bebas (contoh: codingmum_training_git)
      - Referensi: https://www.youtube.com/watch?v=Q3Id0DgcrXY
  • Melakukan commit untuk menambah/merubah file
      • - Buat file baru di repository, index.html
      - Isi dari index.html bisa copy paste dari ini (atau kalau mau bikin sendiri juga boleh): https://github.com/wnitya2/codingmum_training/blob/master/index.html
      - Edit isi dari index.html. Tak masalah kalau belum paham keseluruhan sintaksnya. Coba-coba saja edit isinya :D. Boleh tampilkan biodata sederhana
  • Mempublish website sendiri di GitHub Pages
      • Referensi: https://www.youtube.com/watch?v=BA_c3bGQXlQ
  • Infokan link github dan website yang telah di publish di Group KCMI Newbie
  • Have fun! Mari bermain2 di GitHub
